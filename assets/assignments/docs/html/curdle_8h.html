Detailed Description

The score_record struct definition, plus some useful macro definitions.

Contains the score_record struct, plus macros which allow the program to emit a useful error message before aborting execution.

Data Structure Documentation

◆ score_record

struct score_record

Stores the name and current score of a curdle player.

Initialize a struct score_record with score_record_init().

e.g.:

struct score_record myrec;
score_record_init(&myrec, "carol", 72);

A score_record struct has fixed size and contains no pointers. It may therefore be safely passed to and returned from functions by value.
